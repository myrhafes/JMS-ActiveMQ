# JMS with ActiveMQ
Hi!
This my repository for testing ActiveMQ

## How to install ActiveMQ ?(Windows)
###Step 1: 
From a console window, change to the installation directory and run ActiveMQ:
```
cd [activemq_install_dir]
```
###Step 2: Enter to /bin
```
bin\activemq start
```
